# Leader Website

Modern landing page for the Leader extension, built as a fast static site.

## Files

- `index.html` - page structure and content
- `styles.css` - responsive styling and design system
- `script.js` - smooth-scroll interactions

## Publish

Deploy this folder on any static host (Vercel, Netlify, Cloudflare Pages, GitHub Pages).

## Important setup

Update install buttons in `index.html`:

- Chrome: replace the placeholder Chrome Web Store URL with your Leader listing URL
- Edge: replace the placeholder Edge Add-ons URL with your Leader listing URL
